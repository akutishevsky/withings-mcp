About this health check

This page confirms that the Withings MCP Server is running and ready to accept connections from MCP clients like Claude Desktop. The server provides a secure bridge between your Withings health devices and Claude AI using the Model Context Protocol.

If the status shows OK, you can connect by adding https://withings-mcp.com/mcp as a custom connector in Claude Desktop. The server handles OAuth 2.0 authentication with Withings and encrypts all tokens at rest.

For setup instructions, visit the home page or the GitHub repository.
